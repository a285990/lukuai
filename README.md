# 🚀 LuKuai - 免费AI对话聚合平台

LuKuai 是一个一站式 AI 对话聊天平台，聚合多种主流 AI 模型，让你无需频繁切换网站，即可体验不同 AI 的能力。

---

## ✨ 功能特点

- 🤖 支持 50+ AI 模型  
  包括 ChatGPT、Claude、Gemini、Grok、通义千问等

- 🔄 多模型自由切换  
  一个界面即可体验不同 AI

- ⚡ 高效稳定  
  优化请求速度与响应体验

- 🧠 多场景应用  
  - 写作  
  - 编程  
  - 翻译  
  - 办公  
  - 科研  

- 🌐 国内可用  
  无需复杂配置即可访问
